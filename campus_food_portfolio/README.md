
# 🍔 Campus Food Delivery System (Zomato Style)

A full-stack web application built using Flask that allows students to order food inside campus.

## 🚀 Features
- User Authentication (Login/Register)
- Cart System
- Order Placement & Tracking
- Admin Dashboard
- REST API

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- Bootstrap
- Gunicorn

## 🌐 Deployment
Deployed on Render

## ⚙️ Setup
pip install -r requirements.txt
python app.py
